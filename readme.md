# 摘要
数据集中包含道路工程、轨道工程、公路运输、城市公共交通运输、交通运输经济、交通运输安全等行业提供通识常识。


# 数据：
～450个文本和图片文件，~1.2G

## 1. 科技文献
    a. 科技论文、研究报告  
## 2. 科研基础
    a. 交通科技项目、来自政府部门、科研机构、大专院校和企业等交通运输领域信息  
## 3. 工程建设信息系
    a. 国内交通基础设施（轨道交通基础设施、城市交通基础设施）  
    b. 国外交通基础设施（轨道交通基础设施、城市交通基础设施）  
## 4. 管理决策信息
    a. 交通运输行业发展报告    
    b. 智库报告  
## 5. 其他
    a. 运输统计信息  
    b. 交通基础设施建设知识
